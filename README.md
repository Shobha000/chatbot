 
# Chatbot  

A desktop AI chatbot built using Python and Tkinter. Features include:

- 🗨️ GUI chat interface
- 🎤 Voice input
- 🧠 Memory with toggleable display
- 📅 Date and time answers
- 🤣 Jokes
- ➗ Math calculations
